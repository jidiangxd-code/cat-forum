# 发布页 (Publish) — 页面设计文档

> 版本：v1.0 | 更新日期：2026-04-19

## 一、页面概述

**页面路径**：`/pages/publish/publish`

**页面定位**：用户发布新的校园猫咪帖子的页面，支持上传图片、填写标题和描述、选择标签和位置。

## 三、组件详细设计

### 3.1 导航栏

| 属性 | 值 |
|------|-----|
| 背景 | `#FFFFFF` |
| 标题 | "发布帖子"，`32rpx`, `#333333` |
| 取消按钮 | 左侧，`28rpx`, `#999999` |
| 发布按钮 | 右侧，`28rpx`, `#FF9500`，**满足条件时高亮** |

**发布按钮状态**：
- 未满足条件（无图片/无标题）：`#999999`，不可点击
- 满足条件：`#FF9500`，可点击

### 3.2 图片上传区

```css
.upload-area {
  padding: 24rpx;
  background: #FFFFFF;
  margin-bottom: 16rpx;
}
```

#### 图片网格

```css
.image-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 16rpx;
}
```

| 属性 | 值 |
|------|-----|
| 单图尺寸 | `200rpx × 200rpx` |
| 圆角 | `8rpx` |
| 最多图片 | 9 张 |
| 添加按钮 | 与图片同尺寸，虚线边框 `2rpx dashed #E5E5EA`，中间 `+` 号 |

#### 上传图片交互
- 点击 `+`：调用 `wx.chooseMedia` 选择图片
- 点击图片：全屏查看，支持删除（右上角 🗑️ 图标）
- 长按图片：拖拽排序
- 上传中：显示加载进度环形指示器
- 上传失败：图片变暗，显示 ⚠️ 图标，点击重试

### 3.3 标题输入框


| 属性 | 值 |

```css

  padding: 24rpx;
  background: #FFFFFF;
  margin-bottom: 16rpx;
```

- 支持自定义输入位置
- 支持调用微信位置 API 获取当前位置

## 四、交互说明

| 操作 | 反馈 |
|------|------|
| 点击取消 | 弹出确认框 "确定要放弃编辑吗？" |
| 图片上传完成 | 图片缩略图出现在网格中，显示上传进度 |
| 点击已选图片 | 全屏查看，右上角显示删除按钮 |
| 点击标签选择器 | 弹出标签选择面板 |
| 点击位置选择器 | 弹出位置选择面板 |
| 点击发布（满足条件） | 显示 loading，发布成功后跳转详情页 |
| 点击发布（不满足条件） | 抖动动画 + Toast 提示 "请至少上传一张图片和填写标题" |
| 字数超限 | 输入框变红 + 抖动 + 停止输入 |

## 五、表单验证规则

| 字段 | 规则 | 错误提示 |
|------|------|----------|
| 图片 | 至少 1 张 | "请至少上传一张猫咪照片" |
| 标题 | 必填，2~50 字 | "标题需要 2-50 个字符" |
| 描述 | 选填，最长 500 字 | — |
| 标签 | 至少 1 个 | "请选择至少一个标签" |
| 位置 | 至少 1 个 | "请选择猫咪所在位置" |

## 七、数据来源

| 数据项 | 来源 |
|--------|------|
| 图片上传 | 微信 API `wx.chooseMedia` + 云存储 |
| 标签列表 | 云函数 `getTagList` |
| 位置列表 | 本地配置 + 微信 API `wx.getLocation` |
| 发布接口 | 云函数 `createPost` |
