> 版本：v1.0 | 更新日期：2026-04-19

## 一、页面概述

**页面路径**：`/pages/profile/profile`

**页面定位**：用户的个人中心页面，展示个人信息、发布的帖子、收藏、消息和设置入口。

## 四、交互说明

| 操作 | 反馈 |
|------|------|
| 点击头像 | 弹出更换头像选项（拍照/相册） |
| 点击编辑资料 | 跳转编辑资料页 |
| 点击统计数字 | 跳转到对应列表（发布/收藏/点赞） |
| 点击帖子列表项 | 跳转帖子详情页 |
| 点击"查看全部" | 跳转我的帖子列表页 |
| 点击菜单项 | 跳转对应页面或弹出弹窗 |
| 下拉页面 | 刷新个人信息和数据 |

## 五、子页面设计

### 5.1 编辑资料页

```
┌──────────────────────────────────────┐
│  [←]     编辑资料        [保存]     │
├──────────────────────────────────────┤
│  头像    [头像]  >                  │
│  昵称    [输入昵称...]               │
│  简介    [输入个人简介...]            │
│  位置    [选择学校...]               │
└──────────────────────────────────────┘
```

### 5.2 我的收藏页

- 布局与首页类似，但只显示收藏的帖子
- 支持取消收藏操作

### 5.3 我的点赞页

- 列表展示我点赞过的帖子
- 支持取消点赞操作

## 八、数据来源

| 数据项 | 来源 |
|--------|------|
| 用户信息 | 微信授权登录 `wx.getUserProfile` |
| 统计数据 | 云函数 `getUserStats` |
| 我的帖子 | 云函数 `getUserPosts` |
| 收藏列表 | 云函数 `getUserFavorites` |
| 点赞列表 | 云函数 `getUserLikes` |
