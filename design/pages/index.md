# 首页 (Index) — 页面设计文档

> 版本：v1.0 | 更新日期：2026-04-19

---

## 一、页面概述

**页面路径**：`/pages/index/index`

**页面定位**：用户进入小程序的首页，展示校园猫咪帖子列表，支持浏览、搜索、筛选和快速操作。

## 三、组件详细设计

### 3.1 顶部导航栏

| 属性 | 值 |
|------|-----|
| 背景 | 渐变 `#FF9500 → #FF6B00` |
| 高度 | `96rpx` (不含状态栏) |
| 标题文字 | `36rpx`, `#FFFFFF`, `font-weight: 700` |
| 搜索图标 | 右侧，点击跳转搜索页 |
| 消息图标 | 最右侧，显示未读消息数 |

### 3.2 分类 Tab 栏

| 属性 | 值 |
|------|-----|
| 布局 | 水平滚动 (scroll-view) |
| Tab 数量 | 🔥 热门 / 📍 附近 / 🆕 最新 |
| 激活态 | 底部橙色线条 `4rpx`，文字 `#FF9500`，`font-weight: 600` |
| 默认态 | 文字 `#999999` |
| 高度 | `72rpx` |
| 背景 | `#FFFFFF`，底部阴影 `0 2rpx 8rpx rgba(0,0,0,0.04)` |

### 3.3 帖子卡片

#### 整体结构

```css
.card {
  background: #FFFFFF;
  border-radius: 16rpx;
  margin: 0 32rpx 24rpx;
  padding: 0;
  box-shadow: 0 2rpx 8rpx rgba(0,0,0,0.06);
  overflow: hidden;
}
```

#### 封面图区

```css
.card-image {
  width: 100%;
  height: 400rpx;
  object-fit: cover;
  border-radius: 0; /* 卡片圆角由外层控制 */
}
```

- 占位图：使用默认猫咪占位图 `assets/images/placeholder-cat.png`
- 多图时：显示第一张 + 右上角角标 `+N`

#### 内容区

```css
.card-content {
  padding: 24rpx;
}
```

| 元素 | 样式 |
|------|------|
| 标题 | `30rpx`, `#333333`, `font-weight: 600`, 最多 2 行省略 |
| 标签区 | `margin-top: 12rpx`，水平排列，间距 `8rpx` |
| 互动数据 | `margin-top: 16rpx`，`24rpx`, `#999999` |

#### 标签

| 类型 | 颜色 | 示例 |
|------|------|------|
| 猫咪昵称 | `#FFF3E0` + `#FF9500` | 🏷 小橘 |
| 位置 | `#E8F5E9` + `#34C759` | 📍 食堂附近 |
| 分类 | `#E3F2FD` + `#5AC8FA` | 📷 日常 |

### 3.4 列表项布局

采用**瀑布流**布局（masonry），2列排列：
- 列间距：`16rpx`
- 左右边距：`32rpx`
- 每列卡片宽度：`(750 - 32*2 - 16) / 2 = 335rpx`

> **备选方案**：单列流式布局，适用于"附近"Tab 以地图/位置为主展示。

## 四、交互说明

| 操作 | 反馈 |
|------|------|
| 点击帖子卡片 | 跳转详情页 `/pages/detail/detail?id=xxx` |
| 点击 ❤️ | 点赞，图标变为红色，数字 +1 |
| 长按帖子 | 弹出操作菜单（分享、举报、不感兴趣） |
| 下拉刷新 | 顶部刷新动画，重新加载列表 |
| 上拉触底 | 显示骨架屏加载，追加下一页数据 |
| 点击分类 Tab | 切换分类，重置列表，显示加载动画 |
| 点击搜索图标 | 跳转搜索页 |
| 点击消息图标 | 跳转消息中心 |

## 六、响应式适配

| 屏幕宽度 | 适配方案 |
|----------|----------|
| ≤ 375px (iPhone SE) | 瀑布流列宽适当缩小，图片高度降低至 320rpx |
| 375px ~ 414px (标准) | 默认布局 |
| ≥ 414px (大屏) | 增大卡片间距和图片高度 |
