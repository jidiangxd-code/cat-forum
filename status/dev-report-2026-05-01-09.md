## 二、本次开发任务 —— P2-2 猫咪健康标签

### 完成内容

为猫咪档案系统新增「健康标签」功能，支持创建/编辑/转正时选择多选标签，猫咪主页展示醒目标签。

### 修改文件

#### 1. `cloudfunctions/createCat/index.js` — **更新**
- 新增 `healthTags` 字段支持（从 `event.healthTags` 读取，默认空数组）

#### 2. `cloudfunctions/updateCat/index.js` — **更新**
- 普通编辑：支持更新 `healthTags` 字段（三种 action 均支持）
- 转正（promote）：支持传入 `healthTags`

#### 3. `miniprogram/pages/create-cat/create-cat.js` — **更新**
- `data.form` 新增 `healthTags: []` 字段
- 新增 `healthTagOptions` 选项数组：
  - 🦄 已绝育 / 💉 已驱虫 / 🤰 怀孕中 / 🏥 生病/受伤 / ⚠️ 需要关注
- 新增 `onHealthTagTap()` 方法：多选逻辑（点击切换选中状态）
- `loadExisting()` 加载已有猫咪时正确回填 `healthTags`
- `submit()` 创建和编辑时均传递 `healthTags` 参数

#### 4. `miniprogram/pages/create-cat/create-cat.wxml` — **更新**
- 在「状态选择」下方新增「健康标签」表单项
- 多选标签样式复用现有 `.tag` 样式
- 选中时高亮（橙色背景），已选数量提示

#### 5. `miniprogram/pages/create-cat/create-cat.wxss` — **更新**
- 新增 `.health-tip` 样式（已选数量提示文字）

#### 6. `miniprogram/pages/cat-home/cat-home.wxml` — **更新**
- 在「状态行」下方新增健康标签展示区域
- 使用 `wx:if="{{cat.healthTags && cat.healthTags.length > 0}}"` 按需显示

#### 7. `miniprogram/pages/cat-home/cat-home.wxss` — **更新**
- 新增 `.health-tags-row` 横向滚动标签容器
- 新增三类标签颜色：
  - `.normal`（已绝育/已驱虫）→ 浅绿色 `#E8F5E9`
  - `.warning`（生病/受伤）→ 浅橙色 `#FFF3E0`
  - `.warn`（需要关注）→ 浅黄色 `#FFFDE7`

#### 8. `miniprogram/pages/promote-cat/promote-cat.js` — **更新**
- `form` 新增 `healthTags` 支持
- 新增 `healthTagOptions` 和 `onHealthTagTap()` 方法
- 转正提交时传递 `healthTags` 参数

#### 9. `miniprogram/pages/promote-cat/promote-cat.wxml` — **更新**
- 在状态选择后新增健康标签多选表单

### 数据结构

```js
// 猫咪档案中的 healthTags 字段
{
  healthTags: ['已绝育', '已驱虫', '怀孕中', '生病/受伤', '需要关注']
  // 最多支持 5 种标签，可多选
}
```
