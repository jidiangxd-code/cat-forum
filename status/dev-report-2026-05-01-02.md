## 二、本次开发执行 — P1-1 搜索功能 ✅

### 开发内容

#### 1. api.js 新增搜索函数

**新增 `searchPosts(keyword)`**
- 按帖子内容关键词模糊搜索
- 限制返回30条，按时间倒序
- 内部用正则处理特殊字符转义

**新增 `searchCats(keyword)`**
- 复用已有的 `searchCatProfiles`，补充 `totalVote` 和 `coverImage` 字段
- 返回带投票数和封面的猫咪卡片数据

**导出更新**
- `module.exports` 新增 `searchPosts`、`searchCats`

#### 2. 新建 `pages/search/search` 搜索页

| 文件 | 内容 |
|------|------|
| `search.json` | 页面配置，标题"搜索"，支持下拉刷新 |
| `search.wxml` | 搜索框 + Tab切换（猫咪/帖子）+ 搜索历史 + 热门搜索 + 结果列表 |
| `search.js` | 搜索逻辑 + 历史管理（本地存储最多10条） + Tab切换 + 跳转逻辑 |
| `search.wxss` | 搜索框样式 + Tab样式 + 猫咪卡片 + 帖子卡片 + 空状态 |

**搜索页功能点：**
- 🔍 顶部搜索框，页面加载自动聚焦
- 🐱 / 📝 Tab 切换猫咪/帖子搜索结果
- 🕐 搜索历史（本地存储，保留10条，支持清空）
- 🔥 热门搜索词（7个预设关键词，点击直接搜索）
- 猫咪卡片展示：头像 + 名字 + 外貌/性别/位置 + 票数
- 帖子卡片展示：猫咪来源 + 分类标签 + 内容摘要 + 时间/点赞/评论数
- 点击结果 → 跳转猫咪主页或帖子详情

#### 3. 首页新增搜索入口

- `index.wxml` 横幅右侧新增 🔍 按钮
- `index.wxss` 新增 `.search-btn` 样式
- `index.js` 新增 `goSearch()` 方法 → `wx.navigateTo('/pages/search/search')`

#### 4. app.json 注册搜索页

- `pages/search/search` 加入 pages 数组
