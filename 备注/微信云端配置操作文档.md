# 微信小程序云端配置操作文档

## 一、云开发环境准备

### 1.1 创建云开发环境
1. 打开微信开发者工具，进入小程序项目
2. 点击左侧菜单栏「云开发」
3. 点击「开通」，创建云开发环境
4. 记录云环境ID（后续配置需要）

### 1.2 配置项目环境
1. 修改 `project.config.json`，添加云环境ID：
```json
{
  "cloudfunctionRoot": "cloudfunctions/",

    "minified": true

    "hidedInDevtools": []

    "baseURL": "",
```

## 二、云函数部署

### 2.1 必须部署的云函数

| 云函数名称 | 功能说明 | 是否必需 |
|-----------|---------|---------|
| `login` | 用户登录，获取openid | ✅ 必需 |
| `createCat` | 创建猫咪档案 | ✅ 必需 |
| `getCatProfileList` | 获取猫咪列表/排行榜 | ✅ 必需 |
| `updateCat` | 更新猫咪信息 | ✅ 必需 |
| `mergeCat` | 合并重复猫咪 | ✅ 必需 |
| `publishPost` | 发布帖子 | ✅ 必需 |
| `addComment` | 添加评论 | ✅ 必需 |
| `deletePost` | 删除帖子 | ✅ 必需 |
| `deleteComment` | 删除评论 | ✅ 必需 |
| `voteCat` | 猫咪投票 | ✅ 必需 |
| `toggleCommentLike` | 评论点赞 | ✅ 必需 |
| `followUser` | 关注/取消关注用户 | ✅ 必需 |
| `checkContent` | 内容安全审核 | ✅ 必需 |
| `init-collections` | 初始化数据库集合 | ⚠️ 首次部署时运行一次 |

### 2.2 部署步骤

**方法一：批量上传（推荐）**
1. 在微信开发者工具中，右键点击 `cloudfunctions` 文件夹
2. 选择「上传并部署：云端安装依赖」
3. 等待所有云函数部署完成

**方法二：单个部署**
1. 右键点击需要部署的云函数
2. 选择「上传并部署：云端安装依赖」
3. 等待部署完成

### 2.3 首次部署后初始化
1. 右键点击 `init-collections` 云函数
2. 选择「上传并运行」
3. 在弹出的窗口中点击「确定」
4. 查看运行结果，确认所有集合创建成功

## 三、云存储配置

### 3.1 存储权限设置
1. 进入云开发控制台 → 存储 → 权限设置
2. 修改安全规则为：
```json
{
  "read": true,
  "write": true
}
```

### 3.2 文件夹结构
系统会自动创建以下文件夹结构：
- `images/` - 帖子图片存储
- `avatars/` - 用户头像存储

### 3.3 图片格式限制
- 支持格式：JPG、PNG、GIF
- 单张图片大小限制：≤ 10MB
- 单帖子最多9张图片

## 四、数据库配置

### 4.1 集合列表

| 集合名称 | 用途 | 说明 |
|---------|------|------|
| `cats_profile` | 猫咪档案 | 新版猫咪数据 |
| `posts` | 帖子 | 帖子内容 |
| `comments` | 评论 | 评论内容 |
| `votes` | 投票记录 | 猫咪投票 |
| `favorites` | 收藏记录 | 用户收藏 |
| `follows` | 关注关系 | 用户关注 |
| `notifications` | 通知 | 系统通知 |
| `users` | 用户信息 | 用户资料 |

### 4.2 数据库权限设置
1. 进入云开发控制台 → 数据库 → 安全规则
2. 修改规则为：
```json
{
  "read": true,
  "write": true
}
```

**注意**：正式环境建议设置更严格的权限规则：
```json
{
  "read": true,
  "write": "auth != null"
}
```

### 4.3 索引创建（可选但推荐）

为提升查询性能，建议创建以下索引：

**posts 集合**：
- `authorId` - 作者ID索引
- `catId` - 猫咪ID索引
- `createTime` - 创建时间索引
- `status` - 状态索引

**comments 集合**：
- `postId` - 帖子ID索引
- `authorId` - 作者ID索引
- `status` - 状态索引

**cats_profile 集合**：
- `catType` - 猫咪类型索引
- `status` - 状态索引
- `totalVote` - 投票数索引

## 五、内容安全配置

### 5.1 开通内容安全服务
1. 进入云开发控制台 → 安全 → 内容安全
2. 点击「开通」
3. 按照指引完成配置

### 5.2 配置说明
- 文字审核：自动审核帖子和评论内容
- 图片审核：自动审核上传的图片
- 敏感内容拦截：自动拦截违规内容

## 七、常见问题排查

### 7.1 云函数部署失败
- 检查网络连接
- 检查云函数代码是否有语法错误
- 检查 `package.json` 是否正确配置

### 7.2 数据库操作失败
- 检查集合是否已创建
- 检查数据库权限设置
- 检查记录ID是否正确

### 7.3 图片上传失败
- 检查存储权限设置
- 检查图片大小是否超限
- 检查网络连接

### 7.4 内容审核失败
- 检查内容安全服务是否开通
- 检查是否有敏感内容
- 检查调用频率是否超限
