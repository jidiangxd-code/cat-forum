# 微信云开发控制台操作指南

## 一、前提条件

1. 已注册微信小程序账号
2. 已安装微信开发者工具
3. 已在微信公众平台开通云开发服务

## 二、云开发环境创建

1. **登录微信公众平台**
   - 访问 https://mp.weixin.qq.com/
   - 使用小程序账号登录

2. **进入云开发控制台**
   - 在左侧菜单中找到「云开发」或者在顶栏找
   - 点击进入云开发控制台

3. **创建云开发环境**
   - 点击「创建环境」按钮
   - 填写环境名称（如：xiao-mao-env）
   - 选择环境类型（建议选择「标准型」）
   - 点击「确定」创建环境
   - 等待环境创建完成（约1-3分钟）

4. **获取环境ID**
   - 环境创建完成后，在云开发控制台首页可以看到环境ID
   - 记录下这个环境ID，后续需要配置到小程序中

## 三、数据库初始化

1. **创建集合**
   - 在云开发控制台左侧菜单中点击「数据库」
   - 点击「创建集合」按钮
   - 依次创建以下集合：
     - `cats_profile`：猫咪档案表
     - `posts`：帖子表
     - `comments`：评论表
     - `votes`：投票表

2. **设置权限**
   - 对于每个集合，点击「权限设置」
   - 建议设置为「仅创建者可写，所有人可读」
   - 这样可以保证数据安全的同时，允许所有用户查看内容

## 四、云函数部署

1. **在微信开发者工具中打开项目**
   - 启动微信开发者工具
   - 导入 `c:\Users\郭旭东\Desktop\xiao_mao` 项目

2. **配置云环境**
   - 在微信开发者工具中，点击顶部菜单「云开发」
   - 在弹出的窗口中选择刚才创建的云环境
   - 点击「确定」

3. **安装云函数依赖**
   - 右键点击 `cloud functions` 目录
   - 选择「在终端中打开」
   - 运行命令：`npm install` 安装依赖

4. **部署云函数**
   - 右键点击每个云函数目录（如 `login`、`createCat` 等）
   - 选择「上传并部署」
   - 等待部署完成
   - 重复此步骤，部署所有云函数

5. **验证部署结果**
   - 在云开发控制台左侧菜单中点击「云函数」
   - 确认所有云函数都已成功部署

## 五、云存储配置

1. **创建存储目录**
   - 在云开发控制台左侧菜单中点击「存储」
   - 点击「创建文件夹」按钮
   - 创建以下文件夹：
     - `cats`：存放猫咪图片
     - `posts`：存放帖子图片

2. **设置存储权限**
   - 点击「权限设置」
   - 建议设置为「所有人可读，仅创建者可写」

## 六、小程序配置

1. **配置环境ID**
   - 打开 `miniprogram/envList.js` 文件
   - 修改为以下内容：
   ```javascript
   const envList = [
     {
       envId: '你的环境ID',
       alias: '默认环境'
     }
   ];
   const isMac = false;
   module.exports = {
     envList,
     isMac
   };
   ```
   - 将「你的环境ID」替换为实际的云开发环境ID

2. **配置云开发初始化**
   - 打开 `miniprogram/app.js` 文件
   - 确保云开发初始化代码正确：
   ```javascript
   if (wx.cloud) {
     wx.cloud.init({
       traceUser: true
     });
     console.log('✅ 云开发初始化完成');
   }
   ```

## 七、测试运行

1. **编译小程序**
   - 在微信开发者工具中点击「编译」按钮
   - 等待编译完成

2. **预览小程序**
   - 点击「预览」按钮
   - 使用微信扫描二维码，在手机上查看小程序

3. **测试核心功能**
   - 测试登录功能
   - 测试创建猫咪档案
   - 测试发布帖子
   - 测试投票功能
   - 测试评论功能

## 八、常见问题排查

1. **云函数调用失败**
   - 检查云函数是否已正确部署
   - 检查云函数权限设置
   - 检查云函数代码是否有错误

2. **数据库操作失败**
   - 检查集合是否已创建
   - 检查集合权限设置
   - 检查数据库操作代码是否正确

3. **图片上传失败**
   - 检查云存储权限设置
   - 检查存储目录是否已创建
   - 检查网络连接

4. **小程序无法登录**
   - 检查云函数 `login` 是否已部署
   - 检查网络连接
   - 检查小程序 AppID 是否正确

## 九、注意事项

1. **环境ID配置**：确保在小程序中正确配置了云开发环境ID
2. **云函数部署**：所有云函数都需要部署后才能使用
3. **权限设置**：合理设置数据库和存储的权限，保证数据安全
4. **依赖安装**：云函数需要安装依赖后才能部署
5. **测试验证**：部署后要充分测试核心功能，确保小程序正常运行

## 十、总结

通过以上步骤，你可以在微信云开发控制台中完成环境搭建、云函数部署、数据库初始化和云存储配置，使校园小猫小程序能够正常运行。

如果遇到问题，可以参考微信云开发官方文档或在微信开发者社区寻求帮助。
