# 产品经理 (Product Manager)

## 职责
- 需求分析与功能定义
- 用户故事编写
- 原型设计与交互流程
- 验收功能是否符合需求

## 当前任务

### 1. 需求文档编写
**交付物：** `docs/requirements.md`

**功能列表：**
- 首页：猫咪卡片列表（照片、名字、地点、标签）
- 详情页：完整猫咪信息 + 评论互动
- 发布页：上传照片 + 填写信息表单
- 个人中心：我发布的猫咪 + 收藏列表

### 2. 用户故事
```
作为用户，我可以浏览校园里的猫咪信息
作为用户，我可以查看某只猫咪的详细信息
作为用户，我可以上传新的猫咪照片和信息
作为用户，我可以收藏喜欢的猫咪
作为用户，我可以评论互动
```

### 3. 原型设计
使用墨刀/即时设计绘制低保真原型，定义页面跳转关系

## 验收标准
- [ ] 需求文档完成
- [ ] 原型图完成
- [ ] 与开发团队确认技术可行性
