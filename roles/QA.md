# 测试工程师 (QA Engineer)

## 职责
- 测试用例编写
- 功能测试与回归测试
- Bug 发现与跟踪
- 性能测试与优化建议

## 当前任务

### 1. 测试用例编写
**交付物：** `docs/test-cases.md`

**测试范围：**
- 功能测试：所有核心功能
- 兼容性测试：不同机型、不同微信版本
- 性能测试：加载速度、图片加载
- 边界测试：网络异常、数据为空

### 2. 测试执行
**测试流程：**
1. 冒烟测试 - 核心功能是否可用
2. 功能测试 - 按测试用例逐项测试
3. 回归测试 - 修复后验证
4. 验收测试 - 产品验收

### 3. Bug 跟踪
**Bug 记录格式：**
```markdown
## Bug #001
- **标题：** 首页猫咪列表图片加载失败
- **严重程度：** 高
- **复现步骤：** 1. 打开首页 2. 下滑刷新
- **预期结果：** 图片正常加载
- **实际结果：** 显示默认占位图
- **环境：** iPhone 13, iOS 15, 微信 8.0
```

**位置：** `docs/bugs.md`

### 4. 测试报告
每个版本发布前输出测试报告

## 验收标准
- [ ] 测试用例覆盖所有功能
- [ ] Bug 及时记录并跟踪
- [ ] 严重 Bug 修复率 100%
- [ ] 测试报告完成
